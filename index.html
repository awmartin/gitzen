Gitzen

A better way to use and grok git.

View on GitHub Download .zip Download .tar.gz

Git is a necessary tool for today's developers, but it's often touted as frustrating to grok and use. This stems from a couple of facts:

  1. Git's commands don't mimic what you want to do (e.g. "Change my commit message" is git commit --amend).
  2. Git's conceptual model doesn't mirror typically what you want to do.

gitzen is a set of workflows, scripts, and explanations to help you understand what's going on with git, gain control of your repository, work more efficiently, and get things done confidently.
